# FlappyVersus

Asymmetric multiplayer built for the Discord Activity SDK. Be the last bird standing, or play the Pig and place obstacles to stop the flock!

Feel free to re-use my code or assets as long as you give credit :)

- Client: Phaser + Vite  
- Server: Colyseus + Node.js  
- Platform: Discord Activity SDK

---

## Overview
Fast rounds. Simple inputs. One player places traps, others dodge and survive. Good for small groups inside Discord.

## Run locally
```bash
# clone and install
git clone https://github.com/pdmit/discord-multiplayer-template -b flappy-GM
cd discord-multiplayer-template
npm install

# dev
npm run dev  # start client/server per repo scripts
```
